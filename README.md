# ONE

## Your AI Workforce Awaits

Sell your products, services and ideas freely, at the edge, with your own brand of AI  ... it's surprisingly fast. 

1. Clone this repo

```
git clone https://github.com/one-ie/ONE
```

2. Install

```sh
pnpm install
```

3. Start your server

```sh
pnpm run dev
```
